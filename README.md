# Lumen SDK

A verifiable event ledger and integrity framework.

## Features

- Cryptographic event journaling
- Merkle integrity verification
- State history tracking
- Audit verification
- Resiliency and chaos testing
- Multi-node replication experiments

## Structure

core/
- sqlite_engine.py : ledger storage engine
- merkle.py : Merkle verification
- verifier.py : forensic validation
- lumen_live_network.py : network layer
